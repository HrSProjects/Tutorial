## طريقة تشغيل البوت ▶️ و طريقة اضافة الداتا 🗄️

1. **اول شي تسوي توكن للبوت حقك و بعدها تروح عند ملف config.json**
2. **تحط توكن البوت حقك مكان كلمة BOT_TOKEN**
3. **تشغل البوت `node .`**

`> config.json`

```json
{
  "token": "BOT_TOKEN", // غير مكان ال Bot_TOKEN الى توكن البوت حقك
  "prefix": "!"
}
```

- **طريقة اذافة الداتا:**

1. **اول شي تروح لهذا الامتداد في البروجكت `Database > connection.js`**
2. **تضيف رابط الداتا حقتك و هي رابط MongoDB و اذا ما عندك تقدر تتابع المقطع من [هنا](https://youtu.be/Hue7Li0XWNA?si=mFQ7dQOWuMFme7vW)**
3. **تضيفة بتعريف الـ const اللي موجود عندك**

`> Database > connection.js`

```js
const MongoDBURL = "ضيف رابط الداتا حقتك هنا";
```
